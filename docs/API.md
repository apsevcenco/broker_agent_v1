# API

Base URL: `/api`

- `GET /dashboard/summary`
- `GET /inbox`
- `POST /inbox/message`
- `POST /inbox/:id/classify`
- `POST /inbox/:id/suggest-reply`
- `GET /leads`
- `POST /leads`
- `PATCH /leads/:id`
- `POST /leads/:id/score`
- `GET /tasks`
- `POST /tasks`
- `PATCH /tasks/:id`
- `GET /approvals`
- `POST /approvals/:id/approve`
- `POST /approvals/:id/reject`
- `GET /memory`
- `POST /memory`
- `PATCH /memory/:id`
- `GET /knowledge`
- `POST /knowledge`
- `PATCH /knowledge/:id`
- `DELETE /knowledge/:id`
- `GET /activity`
