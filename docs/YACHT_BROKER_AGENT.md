# Yacht Broker Agent

The Yacht Broker Agent is the first active module in Luxury Mobility AI Operating System.

Responsibilities:

- Yacht brokerage
- Off-market deals
- Distressed yacht opportunities
- Buyer qualification
- Seller qualification
- Broker cooperation
- Lead scoring
- Draft replies
- Approval-controlled disclosures
- Yacht-specific knowledge and memory suggestions

The agent must not send external messages automatically in V1. It prepares drafts, recommendations and tasks for admin approval.
