# Car Rental Agent

The Car Rental Agent is planned, not fully built in V1.

Future capabilities:

- Fleet database
- Vehicle profiles
- Daily, weekly and monthly prices
- Deposits
- Insurance notes
- Included kilometers
- Extra kilometer price
- Driver and chauffeur pricing
- Airport transfers
- Weddings and events
- Delivery and pickup
- Availability calendar
- Rental contract drafts
- Client qualification
- Luxury car sourcing
- Economy, premium, luxury and supercar segments

V1 includes the planned agent definition, `/agents/car-rental`, shared assets support and vehicle-oriented schema notes.
